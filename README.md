1. Clone the respository
2. Rename example.env to .env and fill in. You can get a free postgress db url from **[neon.tech](https://console.neon.tech).** You can **[contact me](https://api.giftedtech.co.ke/contact)** for mpesa integration and bulk sms solutions
3. Install packages using: **```bun install```** or **```npm install```** or **```pnpm install```** or **```yarn install```**
4. Push db: **```npm run db:push```**
5. Build the app: **```npm run build```** or run your dev server: **```npm run dev```**
6. Start the app: **```npm start```**
- Visit your app from **```https://localhost:port```** or **```https://ypur-server-ip:port```** your set port

- Note that the first user to sign up on your app becomes the **```superadmin```** automatically and oversees full control of the app
